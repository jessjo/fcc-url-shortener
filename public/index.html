Free Code Camp URL Shortener Microservice

To use this url shortener append a url to the name.

Ex: https://url-shortener-jessjo.c9users.io/http://www.freecodecamp.com

This will return a new database entry with a short URL ex: {"url":"http://www.freecodecamp.com","new-url":"https://url-shortener-jessjo.c9users.io/121"}

Use the new URL to quickly get redirected to your original page.

If you haven't formatted your request as a short URL or a full URL, you'll get an error.
